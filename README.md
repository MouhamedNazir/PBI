#  TP Power BI – Nettoyage et préparation des données

##  Objectif du TP

Ce TP a pour objectif de manipuler, nettoyer et transformer des données à l’aide de Python (Jupyter Notebook) afin de les exploiter dans Power BI.

---

##  Outils utilisés

* Python
* Jupyter Notebook
* pandas
* Power BI

---

##  Contenu du projet

### 🔹 Notebooks

* `BI_1_en_seance_2.ipynb` : travail réalisé en séance
* `EXERCICE1.ipynb` : exercices pratiques
* `pandas_vs_polars.ipynb` : comparaison d’outils

### 🔹 Données

* `MLAIM.csv` : données brutes
* `MLAIM_cleaned.csv` : données nettoyées
* `Etudiants_clean.csv` : données préparées
* `Etudiants.xlsx` : fichier Excel utilisé

### 🔹 Autres fichiers

* `Exercice_1.txt` : réponses ou consignes

---

##  Travaux réalisés

* Chargement des données avec pandas
* Nettoyage des données :

  * gestion des valeurs manquantes
  * suppression des doublons
  * correction des formats
* Transformation des données
* Export des résultats en CSV / Excel
* Préparation pour utilisation dans Power BI
